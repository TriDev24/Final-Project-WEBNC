# Internet-Banking

## How to start Client:

-   Install packages && dependencies

```
    npm i
```

-   Start Client

```
    npm start
```

## How to start Server:

-   Install packages && dependencies

```
    npm i
```

-   Start Server

```
    npm run dev
```

## API URL:

```
    http://localhost:8080/v1.0.0/<resource>
```
